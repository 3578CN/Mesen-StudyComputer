
## Mesen 改造版：步步高软驱一号模拟器

本项目基于 Mesen2，并借鉴了钳工的 VirtuaNES 源码，对步步高（BBK）软驱一号的 Mapper 进行了大量适配与改造。原版 Mesen2 支持跨平台（Windows / Linux / macOS），但我本人尚未在 Linux 上进行过编译测试，目前主要在 Windows 环境下使用和验证。

> 注意：这是预览版，功能可能有限，存在若干 bug。

> 本模拟器对开发者友好：内置强大调试器，支持反汇编查看、脚本调试、断点、内存与寄存器查看等功能。

> 支持平台/格式：NES、SNES、GBA、GB、PCE、SMS、WS 等，包含大量 Mapper 支持。

### 使用方法

1. 解压缩压缩包，得到 `Mesen.exe`。
2. 首次运行时会弹出设置界面，选择文档保存位置（`Mesen.exe` 会自动解包所需的 DLL 等）。默认路径为 `C:\Users\用户\Documents\Mesen2`，也可以选择将文件保存到 `Mesen.exe` 所在目录。
3. 必须使用压缩包内提供的 `bbk_bios10.nes` 加载 BIOS（该文件的文件头已做改动）。
4. 将软盘镜像文件（例如 `00.IMG`）拖放到 Mesen 窗口，即可从 BIOS 启动并引导到软盘上的 bbgdos。
5. 支持的软盘镜像格式：`ima`、`img`（均为 1.44MB 3.5" 格式）。

模拟器右上角菜单栏右键点击可打开软盘管理菜单，打开磁盘管理对话框后，可以对镜像内的文件进行实时操作：新增、删除、替换。可直接将文件拖放到文件列表以新增；选中文件后按键盘 DEL 键可删除；所有修改会在模拟器内实时同步。

### 已完成的功能

1. 实现软驱一号主要 Mapper 的模拟与磁盘操作，支持多种 GAM 游戏，在中文 DOS 环境下经过测试。
2. 将 Mesen2 的语言包完全汉化（原版为英文）。
3. 完成 Windows 下 HUD 提示的 UTF-8 字体渲染（跨平台支持暂未完善）。
4. 新增软盘镜像实时管理功能，运行时可对镜像内文件进行新增/删除/替换，便于开发与调试。
