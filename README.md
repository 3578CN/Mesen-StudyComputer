## Mesen 模拟器（学生电脑版）

    这是一个学生电脑专用的模拟器，基于 Sour Mesen2 的二次开发版本。
    支持步步高、小霸王、裕兴等品牌的学生电脑的 Mapper。
---
Mesen 是一个非常精致的模拟器，拥有 GBA\GB\SMS\NES\SNES\PCE\WS 多种模拟器功能。

---
 目前暂时只支持到 步步高软驱一号的 bios Mapper，其他 Mapper 还在开发中。

 已完成的功能：
    - 软件界面中文化，已全部翻译成简体中文。  
    - 支持步步高软驱一号的 bios Mapper。  
    - 支持 1.44M 3.5" 软盘镜像加载。  
    - 磁盘文件管理功能，可以实时添加删除复制磁盘文件到 PC。  
    - 软驱工作指示灯显示。  
    - HUD 消息翻译成中文。  

### 使用方法：
1. 使用方法很简单，第一次运行会进入简单配置界面，选择文档的保存路径，默认为你的名字下的文档目录，也可以选择当前 Mesen.exe 所在的目录,选择是否保存到桌面，然后确定就进入主界面了。

2. 先拖放你要加载的磁盘镜像到模拟器，支持 ima、img两种1.44M 3.5"软盘镜像。再拖入 bbk_bios10.nes 到模拟器中，即可运行，自动加载软盘镜像，进入学生电脑界面。

3. 然后使用步步高的 dos 命令行开始运行游戏等任务。
